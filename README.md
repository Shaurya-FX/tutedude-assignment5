# tutedude-assignment5

Python Tasks
Description

This project contains simple Python programs for practicing basic concepts like lists and dictionaries.

Task 1: Student Marks

A dictionary stores student names and their marks.

The user enters a student name.

The program shows the marks if the student exists.

If the student does not exist, a message is shown.

Task 2: List Operations

A list of numbers from 1 to 10 is created.

The first five numbers are extracted.

These numbers are reversed.

Both lists are printed.

How to Run

Open the Python file.

Run the program using Python.

Follow the instructions on the screen.

Purpose

These tasks help beginners learn:

Dictionaries

Lists

User input

Basic Python logic
